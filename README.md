# Hill Cipher Decoder
---
Decode encrypted "N-Hill Cipher" messages

## Contributing

### Requirements
---
- Make
- Python 3.12
- Pip >= 21.0.1
- Other requirements: 
    - requirements.txt 
    - requirements-dev.txt (development dependencies)

#### Installing dev environment requirements:
---
```
make install-dev
```

### Testing
---
```
make coverage
```
